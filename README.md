### Installing

1. **Clone the repository**

   ``` 
   git clone https://github.com/NileshDeshmukh09/google-keep.git
   ```

2. **Navigate to the project directory**

    ``` 
    cd google-keep
     ```

3. **Install dependencies**

    ``` 
    npm install 
    ```

4. **Start the development server**

    ``` 
    npm run dev 
    ```
